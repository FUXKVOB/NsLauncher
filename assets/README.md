# Assets

Поместите сюда иконки и изображения:

- `icon.ico` - Иконка приложения (256x256)
- `icon.png` - Иконка PNG (512x512)
- `background.jpg` - Фоновое изображение (опционально)

## Создание иконки

Вы можете использовать онлайн-конвертер для создания .ico файла из PNG:
- https://convertio.co/ru/png-ico/
- https://www.icoconverter.com/

Рекомендуемые размеры: 16x16, 32x32, 48x48, 64x64, 128x128, 256x256
